## ✨ Overview

**Campus Analytics Engine** is a fully functional, terminal-based academic management system. It manages students, courses, attendance, grades, and financial records using a flat-file database architecture.


```
┌─────────────────────────────────────────────────────┐
│                    main.cpp                         │
│              (Menu Router & Entry Point)             │
└──────────┬──────────────────────────────────────────┘
           │
     ┌─────▼──────────────────────────────────────┐
     │              filehandler.cpp                │
     │  (I/O Engine · Parsing · String Utilities)  │
     └─────┬──────────────────┬───────────────────┘
           │                  │
    ┌──────▼──────┐    ┌──────▼──────┐
    │ student_ops │    │ course_ops  │
    │  (M1 · M2)  │    │  (M2 · M3)  │
    └──────┬──────┘    └──────┬──────┘
           │                  │
    ┌──────▼──────────────────▼──────────────────┐
    │    attendance  │   grades   │  fee_tracker  │
    │     (M3)       │   (M4)     │     (M5)      │
    └──────────────────┬─────────────────────────┘
                       │
                ┌──────▼──────┐
                │   reports   │
                │    (M6)     │
                └─────────────┘
```
